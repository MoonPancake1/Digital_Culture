# Арканоид

## Для запуска:

**1. Создать виртуальное окружение:**

Для Linux/MacOS:
```
python3 -m venv venv
```

Для Windows:
```
python -m venv venv
```

**2. Активировать виртуальное окружение:**

Для Linux/MacOS:
```
. venv/bin/activate
```

Для Windows:
```
./venv/Scripts/Activate.exe
```

**3. Установить зависимости**

Для Linux/MacOS:
```
pip install -r req.txt
```

Для Windows:
```
pip install -r req.txt
```

**4. Запустить файл: *game_starter.py***
